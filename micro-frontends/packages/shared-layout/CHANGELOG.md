# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2025-12-26

### Added
- Complete Ant Design theme configuration matching host app
- Export `themeConfig` for use with Ant Design `ConfigProvider`
- Export `ThemeConfig` type from antd

### Changed
- Theme configuration now includes full token and component styles
- Updated README with theme configuration documentation

## [1.0.0] - 2025-12-26

### Added
- Initial release
- Layout, Navbar, and Footer components
- Intelligent navigation system for standalone and MFE modes
- Navigation utilities with host fallback logic
- Cart preview component
- Language switcher component
