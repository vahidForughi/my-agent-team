## Diagrams Skill Briefing

This part (`diagrams`) contains architectural and sequence diagrams, along with related documentation. It serves as a visual guide to the cloud-native e-commerce platform.

### Key Files to Read First:
- `diagrams/AGENT.md`: Provides a comprehensive overview of the diagrams part.
- `diagrams/README.md`: Index and explanation of available diagrams.

### Top Patterns:
- Diagrams are often created using tools like Draw.io (`.drawio` files) or PlantUML (`.puml` files).
- Diagrams typically illustrate the system architecture, component interactions, and data flows.

### Top Gotchas:
- Diagrams can become outdated if not maintained alongside code changes.
- Lack of clear labeling or inconsistent notation can make diagrams difficult to interpret.
