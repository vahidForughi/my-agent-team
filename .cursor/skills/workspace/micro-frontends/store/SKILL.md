# Store Micro-Frontend Skill

This skill provides guidance for working with the Store micro-frontend, a React application focused on displaying product catalogs, individual product details, and handling user interactions related to browsing and purchasing. It integrates with the Host micro-frontend for shared services like authentication and layout, and communicates with the Catalog microservice for data.
