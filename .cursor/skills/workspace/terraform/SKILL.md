This skill provides comprehensive guidance for working with the Terraform part of the codebase.

The Terraform section provisions the AWS infrastructure (VPC, EKS, ECR, databases, message broker, S3, and OpenSearch) for the platform, ensuring a declarative, reproducible, and environment-decoupled cloud footprint.

For a detailed understanding, refer to the [Terraform AGENT.md](terraform/AGENT.md).
