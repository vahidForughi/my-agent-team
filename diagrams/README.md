# Architecture Diagrams

This directory contains architecture and flow diagrams for the platform.

## Viewing

The `.eraserdiagram` files require [Eraser.io](https://eraser.io) to view and edit. You can:

1. Open them directly in the [Eraser.io editor](https://app.eraser.io)
2. View rendered architecture images in the [`images/`](../images/) directory

## Contents

Key diagrams include:
- System architecture and service topology
- CQRS and event-driven patterns
- CI/CD pipeline flows
- Deployment topology (K8s, AWS EKS)
- Micro-frontend composition
- Data flow and saga patterns
